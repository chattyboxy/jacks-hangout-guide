Moneyreact Guide:

Moneyreact is a channel where you can react on a message for Coins!

Once a day you're able to react to a message in #money-react to be able to get coins!

There is 2 possible coins amounts you can get: 500 and a jackpot amount which is 1000

If you want to get a ping every time it resets, just go to the #commands channel and type !moneyreact
